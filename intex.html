Project για το τμήμα της Javascript

Σκοπός του Project είναι να δημιουργηθεί ένα πρόγραμμα το οποίο να λειτουργεί σαν multiple choice exam (εξέταση πολλαπλών επιλογών).

Το project θα αντλεί, στο πρώτο του στάδιο, τις ερωτήσεις απο ένα .json αρχείο (δείγμα με δύο εγγραφές στο πορτοκαλί πλαίσιο). Στη συνέχεια θα το κάνουμε να αντλεί τα δεδομένα απο ένα nodejs σερβερ αλλά αυτό δεν θέλουμε να υλοποιηθεί προς το παρόν.

Οι ερωτήσεις θα πρεπει να εμφανίζονται με τη σειρά και στο τέλος να εμφανίζεται το σκορ μας, σε πόσες ερωτήσεις απαντήσαμε σωστά.

Οι ερωτήσεις θα πρεπει δημιουργούνται απο την javascript και όχι να υπάρχουν στην html τόσο για την εξυπηρέτηση της εκμάθησης της javascript οσο και για να ειναι το πρόγραμμά μας δυναμικό και να προσαρμόζεται σε όποιες και όσες ερωτήσεις θέλουμε αλλάζοντας μόνο το .json αρχείο.
